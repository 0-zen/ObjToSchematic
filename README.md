# Usage
* Download and install [Node.js](https://nodejs.org/en/).
* Execute `git clone https://github.com/LucasDower/ObjToSchematic.git` in your command line.
* Navigate to `/ObjToSchematic-main`.
* Run `npm install`.
* Run `npm start`.
